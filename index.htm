Later on in Middle School I would start learning python as one of my first forays into proper coding languages, using it mostly to write simple input output text handling scripts and still consider python one of the best tools for quick projects.
High school saw my first forray into object based languages with Java and also my first introduction to webdesign with HTML, CSS and, JavaScript.
